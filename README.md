# 개인 프로젝트
## 사용 스택
- Backend : Spring Boot
- Frontend : Next js, React
## 기능 구현
- 로그인 및 유저정보 CRUD
- 네이버 최신 뉴스 크롤링
- Q&A 게시판 (유저의 질문과 운영자의 답변)
